<div>&lt;p&gt;DataDirect SequeLink provides access to data using ODBC drivers. SequeLink can be used to access data that has a Windows ODBC driver; however, does not have a UNIX ODBC driver. Use SequeLink to access data that resides on a Windows machine from a UNIX, or Linux machine that runs PowerCenter Services. SequeLink provides ODBC drivers for databases such as, Microsoft Access, Filemaker, Progress, FoxPro, and dBase.&lt;/p&gt;</div>
<div>&lt;p&gt;&lt;strong&gt;Note&lt;/strong&gt;&lt;/p&gt;</div>
<div>&lt;p&gt;The SequeLink ODBC drivers that Informatica provides can only be used with PowerCenter.&lt;/p&gt;</div>
